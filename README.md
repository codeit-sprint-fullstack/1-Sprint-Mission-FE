# 📑 스프린트 미션5 | 

스프린트 미션 5는 React를 사용하여 요구사항에 따라 중고마켓 페이지를 구현하였습니다.

### 👨‍👩‍👧‍👦 요구사항 

- React, API를 사용하여 PC, Tablet, Mobile 디자인에 해당하는 중고마켓 페이지를 만들고, 페이지네이션 기능과 상품 목록 검색 기능을 구현합니다.
